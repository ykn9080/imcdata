# imcData

imcData enable to create various type of data easily.

## Installation

Use npm or yarn.

```bash
npm install imcData
```

## Usage

```js
import Data from "imcData"

# send 'data' and return 'onChange'
<Data data={data} onChange={onChange} />

const onChange=(val)=>{
//val={dtlist:[...], dtsetting={}}
}
```
